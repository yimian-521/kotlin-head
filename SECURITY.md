# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in kotlin-head, please **do not** open a public GitHub Issue.

Instead, email: **yimian.666@qq.com**

We will respond within 7 days and work with you on a fix and coordinated disclosure.

## Supported Versions

| Version | Supported |
|---------|-----------|
| v0.11.1 | ✅ Active |
| v0.10.x | ✅ Security fixes |
| < v0.10.0 | ❌ End of life |

## Security Design

- Zero external dependencies — no supply chain attack surface
- SHA256 deterministic build for verification
- NIST SP 800-53 Rev 5.2 compliant (21/21 controls)
- Military-grade fault tolerance (178S 10/10)
